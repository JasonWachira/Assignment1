## Jason Wachira Kamwaro 172995 Workflow

### Initialise empty git repository
```bash
git init
```

### Make a main branch and switch to it
```bash
git checkout -b main
```

### Add all files
```bash
git add .
```

### Commit changes
```bash
git commit -m "Initial commit"
```

### Set a remote origin to push to
```bash
git remote add origin "https://github.com/JasonWachira/Assignment1.git"
```

### Push changes to remote repository
```bash
git push -u origin main
```
